# CLO (Course Learning Outcomes)

## ความหมาย

CLO ย่อมาจาก Course Learning Outcomes หมายถึง ผลลัพธ์การเรียนรู้รายวิชา ซึ่งเป็นสิ่งที่นักศึกษาคาดหวังว่าจะสามารถทำได้ภายหลังจากการเรียนรู้ในรายวิชานั้นๆ เสร็จสิ้น

## ลักษณะของ CLO ที่ดี

CLO ที่ดีควรมีลักษณะดังนี้:

1. **Specific (เฉพาะเจาะจง)** — ระบุได้ชัดเจนว่าผู้เรียนจะทำอะไรได้
2. **Measurable (วัดได้)** — สามารถประเมินได้
3. **Achievable (บรรลุได้)** — สมเหตุสมผลกับระดับของรายวิชา
4. **Relevant (เกี่ยวข้อง)** — เชื่อมโยงกับ PLO ของหลักสูตร
5. **Time-bound (กำหนดเวลา)** — ชัดเจนว่าเมื่อไหร่จะบรรลุ

## การเขียน CLO

รูปแบบที่นิยมใช้คือ ABCD:
- **A**ction Verb — คำกริยาที่แสดงการกระทำ (เช่น อธิบาย, คำนวณ, ออกแบบ)
- **B**ehavior — พฤติกรรมที่ต้องการให้แสดงออก
- **C**ondition — เงื่อนไขหรือบริบทที่จะให้ทำ
- **D**egree — ระดับมาตรฐานที่ยอมรับได้

## ตัวอย่าง CLO

| รายวิชา | CLO |
|---------|-----|
| การเขียนโปรแกรมคอมพิวเตอร์ 1 | สามารถเขียนโปรแกรมภาษา Python เพือแก้ปัญหาทางคณิตศาสตร์ได้ |
| โครงสร้างข้อมูล | สามารถเลือกใช้โครงสร้างข้อมูลที่เหมาะสมกับปัญหาได้ |
| การเรียนรู้ของเครื่อง | สามารถใช้งาน library สำหรับ machine learning ได้ |

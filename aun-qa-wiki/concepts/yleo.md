# YLO — Year Learning Outcomes

> YLO (Year Learning Outcomes) เป็นผลการเรียนรู้ที่ผู้เรียนควรบรรลุหลังเสร็จสิ้นแต่ละปีการศึกษา
> หลักสูตรวิศวกรรมคอมพิวเตอร์และปัญญาประดิษฐ์ มรอ. **ไม่ได้กำหนด YLO ไว้อย่างเป็นทางการ**
> แต่สามารถ derive ได้จากการรวม CLO ของรายวิชาที่เรียนในแต่ละปี
> ดูรายละเอียดครบถ้วนใน [[summaries/clo-ylo-mapping]]

## โครงสร้างความสัมพันธ์

```
PEO  →  PLO1–PLO5  →  YLO1 / YLO2 / YLO2b
                         ↑
              CLO (รายวิชา แต่ละ CLO → PLO)
```

---

## YLO1 — ผลการเรียนรู้ปีที่ 1 (ฐานความรู้และทักษะ AI/ML + วิจัย)

**ภาคการศึกษาที่รองรับ:** ปีที่ 1 ภาค 1 และ ปีที่ 1 ภาค 2

**PLOs ที่ครอบคลุม:** PLO1, PLO2, PLO3, PLO4, PLO5 (ครบทุก PLO)

**ผลการเรียนรู้ที่คาดหวัง:**
หลังสิ้นปีที่ 1 ผู้เรียนสามารถ:
- อธิบาย/ประยุกต์/วิเคราะห์หลักการ AI & ML (PLO1, PLO2)
- เขียนโปรแกรม ML และแก้ปัญหาเชิงวิศวกรรม (PLO1)
- ออกแบบงานวิจัยตามระเบียบวิธีวิทยา (PLO3)
- ติดตามและประเมินเทคโนโลยีใหม่อย่างมีจริยธรรม (PLO4, PLO5)
- นำเสนอบทความวิจัยในสัมมนาได้ (PLO5)

**รายวิชาในปีที่ 1:**

| ภาค | รหัส | ชื่อวิชา |
|-----|------|---------|
| ภาค 1 | 7015101 | ปัญญาประดิษฐ์และการเรียนรู้ของเครื่อง |
| ภาค 1 | 7015906 | สัมมนาทางวิศวกรรมคอมพิวเตอร์และปัญญาประดิษฐ์ 1 |
| ภาค 1 | 7015907 | ระเบียบวิธีวิจัยทางวิศวกรรมคอมพิวเตอร์และปัญญาประดิษฐ์ |
| ภาค 2 | 7015102 | วิทยาการข้อมูลและวิศวกรรมข้อมูล |
| ภาค 2 | 7015103 | การเรียนรู้เชิงลึกและการประมวลผลภาพ |
| ภาค 2 | 7015908 | สัมมนาทางวิศวกรรมคอมพิวเตอร์และปัญญาประดิษฐ์ 2 |

## YLO2 — ผลการเรียนรู้ปีที่ 2 แผน 1 (วิจัย/นวัตกรรมเชิงลึก — วิทยานิพนธ์)

**ภาคการศึกษาที่รองรับ:** ปีที่ 2 ภาค 1, 2, 3 (แผน 1)

**PLOs ที่ครอบคลุม:** PLO1, PLO2, PLO3, PLO4, PLO5

**ผลการเรียนรู้ที่คาดหวัง:**
หลังสิ้นปีที่ 2 (แผน 1) ผู้เรียนสามารถ:
- ออกแบบและดำเนินงานวิจัยเชิงลึกด้าน AI/CS อย่างอิสระ (PLO3)
- สร้างนวัตกรรมหรือองค์ความรู้ใหม่ในสาขา (PLO1, PLO3)
- เขียนและตีพิมพ์บทความวิชาการระดับชาติ/นานาชาติ (PLO5)
- นำเสนอผลงานวิทยานิพนธ์และสอบป้องกัน (PLO4, PLO5)

**รายวิชาหลัก:**

|-----|------|---------|
| ภาค 1 | 7015901 | วิทยานิพนธ์ 1 |
| ภาค 2 | 7015902 | วิทยานิพนธ์ 2 |
| ภาค 3 | 7015903 | วิทยานิพนธ์ 3 |

## YLO2b — ผลการเรียนรู้ปีที่ 2 แผน 2 (พัฒนานวัตกรรมประยุกต์ — สารนิพนธ์)

**ภาคการศึกษาที่รองรับ:** ปีที่ 2 ภาค 1, 2 (แผน 2)

**PLOs ที่ครอบคลุม:** PLO1, PLO2, PLO3, PLO4, PLO5

**ผลการเรียนรู้ที่คาดหวัง:**
หลังสิ้นปีที่ 2 (แผน 2) ผู้เรียนสามารถ:
- นำความรู้ AI/CS ไปพัฒนาระบบ/ผลิตภัณฑ์ประยุกต์ใช้งานจริง (PLO1, PLO2)
- วิเคราะห์ข้อมูลและประเมินผลโครงการอย่างเป็นระบบ (PLO2, PLO3)
- นำเสนอสารนิพนธ์และสอบป้องกัน (PLO4, PLO5)

**รายวิชาหลัก:**

|-----|------|---------|
| ภาค 1 | 7015904 | สารนิพนธ์ 1 |
| ภาค 2 | 7015905 | สารนิพนธ์ 2 |

## PLO Coverage สรุป (ทุก YLO)

| PLO | ความหมาย | YLO1 | YLO2 | YLO2b |
|-----|----------|:----:|:----:|:-----:|
| PLO1 | สร้างสรรค์ผลงาน/นวัตกรรมด้วยซอฟต์แวร์/ฮาร์ดแวร์ | ✓ | ✓ | ✓ |
| PLO2 | ประยุกต์ใช้ความรู้แก้ปัญหาองค์กร/ชุมชน | ✓ | ✓ | ✓ |
| PLO3 | สร้างองค์ความรู้/ต้นแบบนวัตกรรมด้าน AI | ✓ | ✓ | ✓ |
| PLO4 | จรรยาบรรณวิชาชีพ | ✓ | ✓ | ✓ |
| PLO5 | คิดวิเคราะห์/สื่อสาร/ทำงานร่วมกัน/เรียนรู้ตลอดชีวิต | ✓ | ✓ | ✓ |

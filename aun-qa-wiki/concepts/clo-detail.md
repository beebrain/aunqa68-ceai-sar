## รายละเอียด CLO วิชา 7015101 (ตัวอย่าง)

### ปัญญาประดิษฐ์และการเรียนรู้ของเครื่อง (Artificial Intelligence and Machine Learning)

**หน่วยกิต:** 3(2-2-5)  
**กลุ่มวิชา:** วิชาเฉพาะด้านบังคับ  
**แผนการศึกษา:** ปีการศึกษา 2568 ภาคการศึกษาที่ 1

#### คำอธิบายรายวิชา
พื้นฐานของปัญญาประดิษฐ์และการเรียนรู้ของเครื่อง การแก้ปัญหาด้วยการค้นหา การจัดการและการวิเคราะห์ข้อมูล การเรียนรู้แบบมีผู้สอนและไม่มีผู้สอน เทคนิคการหาค่าเหมาะสมและการลดค่าความชันเชิงเกรเดียนต์ โครงข่ายประสาทเทียมและการเรียนรู้เชิงลึก การวัดประสิทธิภาพและปรับปรุงแบบจำลอง การประยุกต์ใช้งานจริงภายใต้หลักจริยธรรมและความรับผิดชอบต่อสังคม

#### CLOs (ผลการเรียนรู้รายวิชา)

| CLO | Verb (การกระทำ) | Object (สาระ) | ส่วนขยาย | Bloom's |
|-----|-----------------|---------------|-----------|---------|
| CLO1 | อธิบาย | หลักการพื้นฐานของปัญญาประดิษฐ์และการเรียนรู้ของเครื่อง ทั้งแบบมีผู้สอนและไม่มีผู้สอน | ได้อย่างถูกต้อง | U (Understand) |
| CLO2 | ประยุกต์ใช้ | เทคนิคการแก้ปัญหาด้วยการค้นหาและการวิเคราะห์ข้อมูล | ในการพัฒนาแบบจำลองการเรียนรู้ของเครื่องได้ | Ap (Apply) |
| CLO3 | วิเคราะห์ | ประสิทธิภาพของแบบจำลองโครงข่ายประสาทเทียมและการเรียนรู้เชิงลึก | เพื่อเลือกใช้และปรับปรุงให้เหมาะสมกับปัญหาที่กำหนดได้ | An (Analyze) |
| CLO4 | แสดงออก | ความตระหนักในจริยธรรมและความรับผิดชอบต่อสังคม | ในการพัฒนาและประยุกต์ใช้ระบบปัญญาประดิษฐ์ | Va (Value) |
| CLO5 | พัฒนา | ระบบปัญญาประดิษฐ์ | โดยใช้เทคนิคการเรียนรู้ของเครื่องที่เหมาะสมในการแก้ปัญหาจริงได้อย่างมีประสิทธิภาพ | A (Articulation) |

#### ความเชื่อมโยง CLO-PLO

| CLO  | Bloom's Level | PLO        |
| ---- | ------------- | ---------- |
| CLO1 | U             | PLO1       |
| CLO2 | Ap            | PLO1       |
| CLO3 | An            | PLO3       |
| CLO4 | Va            | PLO3       |
| CLO5 | A             | PLO1, PLO3 |

## Bloom's Taxonomy Reference

| Code | Level        | คำอธิบาย          |
| ---- | ------------ | ----------------- |
| U    | Understand   | ความเข้าใจ        |
| Ap   | Apply        | การประยุกต์ใช้    |
| An   | Analyze      | การวิเคราะห์      |
| Va   | Value        | การให้คุณค่า      |
| A    | Articulation | การสื่อสาร/นำเสนอ |
| E    | Evaluate     | การประเมินค่า     |
| Cr   | Create       | การสร้างสรรค์     |
| O    | Obtain       | การได้มา          |
| In   | Integrate    | การบูรณาการ       |
| Re   | Review       | การทบทวน          |
| Rs   | Respond      | การตอบสนอง        |
| OA   | Organize     | การจัดระเบียบ     |
| P    | Present      | การนำเสนอ         |

PLO and CLO Summary

PLO (Program Learning Outcomes) และ CLO (Course Learning Outcomes) ของหลักสูตรวิศวกรรมคอมพิวเตอร์และปัญญาประดิษฐ์ บัณฑิตศึกษา

PLOs (ผลลัพธ์การเรียนรู้ที่คาดหวังของหลักสูตร)

PLO ความหมาย ภาษาอังกฤษ
PLO1 การสร้างสรรค์ผลงาน/นวัตกรรม Create
PLO2 การประยุกต์ใช้ความรู้แก้ปัญหา Apply
PLO3 การสร้างองค์ความรู้/ต้นแบบ Create Knowledge
PLO4 จรรยาบรรณวิชาชีพ Ethics
PLO5 การคิดวิเคราะห์/สื่อสาร/ทำงานร่วมกัน/เรียนรู้ตลอดชีวิต Analyze/Communicate/Collaborate/Lifelong

PLO Details (แบบละเอียด)

PLO1: ใช้ซอฟต์แวร์หรือฮาร์ดแวร์ทางคอมพิวเตอร์ในการสร้างสรรค์ผลงาน องค์ความรู้ และนวัตกรรม

PLO2: ประยุกต์ใช้ความรู้ในสาขาวิชาวิศวกรรมคอมพิวเตอร์เพื่อแก้ไขปัญหาความต้องการขององค์กร ชุมชน และท้องถิ่น

PLO3: สร้างองค์ความรู้ทางด้านวิศวกรรมคอมพิวเตอร์และปัญญาประดิษฐ์ที่ตอบสนองความต้องการขององค์กร (แผน 2: สร้างต้นแบบนวัตกรรม)

PLO4: ปฏิบัติตามหลักจรรยาบรรณวิชาชีพวิศวกรรมคอมพิวเตอร์

PLO5: สามารถคิดวิเคราะห์ การสื่อสาร การทำงานร่วมกัน การปรับตัว และการเรียนรู้ตลอดชีวิต

Criterion 4 — Student Assessment

Criterion 4 — Student Assessment

ที่มา: เอกสารอบรม ผศ.ดร.ขวัญดาว (25 มี.ค. 2569)

Requirements Overview

ข้อ ข้อกำหนด (สรุป)
4.1 วิธีประเมินหลากหลาย และ Constructive Aligned กับ CLOs/PLOs
4.2 นโยบายการประเมินและการอุทธรณ์ชัดเจน สื่อสารนศ. ใช้คงเส้นคงวา
4.3 มาตรฐาน/ขั้นตอนความก้าวหน้าและสำเร็จการศึกษา ชัดเจน สื่อสาร ใช้คงเส้นคงวา
4.4 วิธีประเมินมี Rubrics, Marking Schemes, Timelines, Regulations — ต้องมี Validity, Reliability, Fairness
4.5 วิธีประเมินวัดการบรรลุ CLOs และ PLOs ได้จริง (Method + Achievement Criteria)
4.6 Feedback ผลการประเมินให้ทันเวลา ทุกรายวิชา
4.7 ระบบทบทวนการประเมินอย่างต่อเนื่อง มีบุคคลที่ 3 ร่วม ทุกวิชา

R 4.1 — วิธีประเมินหลากหลาย + Constructive Aligned

เนื้อเกณฑ์ที่พึงพิจารณา:

  1. ทุกวิชา มีวิธีการประเมินผู้เรียนที่ หลากหลาย
  2. Assessment ทุกวิชา ต้องวัดและประเมิน CLOs โดยนำ CLOs มาเป็นตัวออกแบบวิธีประเมิน (ทุกรายวิชา)

ตัวอย่าง:
- CLO "จงอธิบาย..." → วิธีวัด: สอบ paper, take home, สอบปากเปล่า
- CLO "สามารถทำ...ได้" → วิธีวัด: ปฏิบัติ หรือ ปฏิบัติร่วมกับ paper

⚠️ วัตถุประสงค์ = มุมมองอาจารย์ | CLOs = มุมมองนักศึกษา

R 4.2 — นโยบายการประเมินและการอุทธรณ์

เนื้อเกณฑ์ที่พึงพิจารณา:

  1. นโยบายการประเมิน: ชัดเจน สื่อสารไปยังผู้เรียน ใช้อย่างคงเส้นคงวา (รายละเอียดการประเมินในรายวิชา: สอบอะไร แบบไหน เงื่อนไขคะแนน)
  2. นโยบายการอุทธรณ์: ชัดเจน สื่อสาร ใช้คงเส้นคงวา
    - ต้องมีระบบมาตรฐาน ชัดเจน ใครทำ (คนกลาง) มีขั้นตอนอย่างไร
    - แยกจากระบบอุทธรณ์อื่นๆ

R 4.3 — มาตรฐานความก้าวหน้าและสำเร็จการศึกษา

เนื้อเกณฑ์ที่พึงพิจารณา:

  • กำหนดมาตรฐาน/เกณฑ์/เงื่อนไข สื่อสาร ใช้คงเส้นคงวา
  • ตัวอย่างมาตรฐาน: เงื่อนไขการเป็นผู้เรียน, การพ้นสภาพ, การขึ้นชั้นปี, เงื่อนไขสำเร็จการศึกษา, เงื่อนไขฝึกงาน, เกณฑ์ผ่านฝึกงาน

สำหรับหลักสูตรบัณฑิตศึกษา (วิจัยอย่างเดียว):
- มาตรฐานและเกณฑ์ผ่าน Thesis แต่ละตัว (Thesis 1 → Thesis 2 → ... ต้องผ่านเงื่อนไขอะไรบ้าง)

R 4.4 — Rubrics, Marking Schemes, Validity, Reliability, Fairness

เนื้อเกณฑ์ที่พึงพิจารณา (ทุกวิชา):

  1. Rubrics — เกณฑ์ประเมินแบบรูบริก (Analytic Scoring Rubric: ระดับคะแนน + เกณฑ์ + คำพรรณนา)
  2. Marking Schemes — กำหนดชัดเจนว่าตอบอะไรได้กี่คะแนน
  3. Timelines — กำหนดระยะเวลาชัดเจน
  4. Regulations — กำหนดกฎข้อบังคับการสอบ
  5. Validity (ความตรง) — ตรงตาม CLOs และ Learning Level
  6. Reliability (ความเที่ยง) — วิชาเดียวกัน เรียนหลายกลุ่ม ใช้ข้อสอบ/วิธีวัดชุดเดียวกัน
  7. Fairness (ความยุติธรรม) — ทุกคน/กลุ่ม ได้รับการประเมินอย่างยุติธรรม (ตัดเกรดแบบอิงเกณฑ์ทุกวิชา)

สำหรับหลักสูตรที่มีวิจัยอย่างเดียว:
- ต้องมีการประเมินคุณภาพวิทยานิพนธ์และ activity อื่นๆ ที่ช่วยผลักดัน PLO

R 4.5 — วัดการบรรลุ CLOs และ PLOs (Method + Achievement Criteria)

เนื้อเกณฑ์ที่พึงพิจารณา:

  • ทุกวิชา: ต้องวัดการบรรลุ CLOs → Method + Achievement Criteria CLOs
  • หลักสูตร: ต้องวัดการบรรลุ PLOs → Method + Achievement Criteria PLOs

⚠️ ความต่าง R4.1 กับ R4.5:
- R4.1 = เน้นวิธีประเมินที่สอดคล้องกับ CLOs (ดูจาก Course Syllabus / มคอ.)
- R4.5 = เน้นachievement (บรรลุหรือไม่) → ต้องมีเกณฑ์ตัดสินว่าผ่านหรือไม่ผ่าน จึงจะวัดการบรรลุ PLOs/CLOs ได้

R 4.6 — Feedback ผลการประเมินทันเวลา

เนื้อเกณฑ์ที่พึงพิจารณา:

  • จุดมุ่งหมาย: ช่วยให้ผู้เรียนวางแผนการเรียนได้ทัน ไม่สอบตกหรือ dropout
  • ต้องบอกวิธีป้อนกลับผลการประเมิน พร้อมระบุช่วงเวลาที่ป้อนกลับ
  • ดำเนินการทุกรายวิชา

R 4.7 — ทบทวนการประเมินอย่างต่อเนื่อง (มีบุคคลที่ 3)

เนื้อเกณฑ์ที่พึงพิจารณา:

ระบบตรวจเช็คการวัดและประเมินผลในทุกวิชา:
1. การวัดประเมินวัด CLOs ได้จริง
2. การวัดประเมิน สอดคล้องกับสภาพจริงในการทำงาน
3. ต้องมี บุคคลที่ 3 เข้ามาร่วมทบทวน

ระบบต้องกำหนดชัดเจน:
- ทำเมื่อไหร่ | ใครทำ | ข้อมูลนำเข้าคืออะไร | พิจารณาอะไร | ปรับปรุงอะไรบ้าง

การทวนสอบ = วิธีหนึ่งในการติดตามว่าการวัดผลเป็นไปตาม CLOs หรือไม่ → ผู้ทวนสอบ feedback อะไร → ผู้สอนนำไปปรับปรุงอย่างไร

PDCA:
- A = ปรับปรุงพัฒนา (ถ้ายังมีจุดพัฒนา)
- A = มาตรฐาน/standard (ถ้าระบบดีอยู่แล้ว → ขยายใช้กับงานอื่น)

Addendum (สำหรับหลักสูตรบัณฑิตศึกษา)

  • Criterion f: "Assessment Methods" รวมถึง semester-based/annual progress monitoring, thesis/dissertation final assessment (viva-voce/thesis defence)

ความสัมพันธ์กับ Criterion อื่น

  • R4.1 ↔ R3.3 (Active Learning ต้องสอดคล้องกับการประเมิน)
  • R4.5 ↔ R1.5 (Method ของ R1.5 = Method ของ R4.5) ↔ R8.4 (Achievement Level)
  • R4.4 Validity ↔ R4.1 (สอดคล้อง CLOs)
  • R4.7 ทบทวน → R3.6 ทบทวน TLA (ต้องมีระบบทั้งสอง)

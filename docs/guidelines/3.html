Criterion 3 — Teaching and Learning Approach

Criterion 3 — Teaching and Learning Approach

ที่มา: เอกสารอบรม ผศ.ดร.ขวัญดาว (25 มี.ค. 2569)

Requirements Overview

ข้อ ข้อกำหนด (สรุป)
3.1 ปรัชญาการศึกษาชัดเจน สื่อสารกับ SHs และสะท้อนใน TLA
3.2 TLA เปิดโอกาสให้ นศ. มีส่วนร่วมอย่างมีความรับผิดชอบ
3.3 TLA เน้น Active Learning ทุกรายวิชา
3.4 TLA ส่งเสริม Lifelong Learning (LLL)
3.5 TLA ปลูกฝัง New Ideas, Creative Thought, Innovation, Entrepreneurial Mindset
3.6 TLA ได้รับการปรับปรุงอย่างต่อเนื่อง สอดคล้องกับภาคการทำงานและ PLOs

R 3.1 — Educational Philosophy

เนื้อเกณฑ์ที่พึงพิจารณา:

  1. กำหนดปรัชญาการศึกษา ของ ม. ให้ชัดเจน (ต้องทำความเข้าใจก่อน)
  2. สื่อสารปรัชญา ไปยัง SHs ทุกกลุ่มที่ต้องรู้:
    - SHs ที่เกี่ยวกับ TLA: ผู้สอน นศ. อ.พิเศษ วิทยากร แหล่งฝึกงาน จนท.
    - กำหนดช่องทางสื่อสารให้เหมาะกับ SHs แต่ละกลุ่ม
  3. นำปรัชญามาสะท้อน ในกิจกรรมการเรียนการสอน

นิยาม: Educational philosophy = set of related beliefs ที่มีอิทธิพลต่อ what & how สอน กำหนดจุดประสงค์การศึกษา บทบาทผู้สอน/ผู้เรียน เนื้อหา และวิธีการ

R 3.2 — Active Participation (นศ.มีส่วนร่วมอย่างมีความรับผิดชอบ)

เนื้อเกณฑ์ที่พึงพิจารณา:

  • TLA ต้องเปิดโอกาสให้ นศ. มีส่วนร่วมในการตัดสินใจ อย่างมีความรับผิดชอบ
  • ไม่จำเป็นต้องทุกรายวิชา
  • ตัวอย่าง: นศ.เลือก อ.ที่ปรึกษาได้เอง, เลือกหัวข้องาน/วิทยานิพนธ์ได้, นศ.มีส่วนประเมินผลงานเพื่อน

R 3.3 — Active Learning ทุกรายวิชา

เนื้อเกณฑ์ที่พึงพิจารณา:

  • ทุกรายวิชา ต้องจัดการเรียนการสอนแบบ Active Learning (เปิดโอกาสให้ผู้เรียนลงมือปฏิบัติจริง)
  • เหตุผล: CLOs ของทุกรายวิชาเป็น Action Verb → นศ.ต้องได้ฝึกทำ CLOs นั้นจริงๆ ผ่าน activity

ตัวอย่าง: CLO "สามารถอธิบาย...ได้" → ต้องมี activity ให้ นศ.ฝึกอธิบาย ไม่ใช่แค่อาจารย์สอนอธิบายอย่างเดียว

Active Learning: นศ.ได้ฝึกซึ่ง CLOs

R 3.4 — Lifelong Learning (LLL)

เนื้อเกณฑ์ที่พึงพิจารณา:

  1. TLA สนับสนุนส่งเสริมให้ผู้เรียน เกิดการเรียนรู้
  2. TLA สนับสนุน เรียนรู้วิธีการเรียนรู้
  3. TLA ปลูกฝัง Lifelong Learning (LLL): critical inquiry, information-processing skills, ทดลองแนวคิดใหม่

สำคัญ:
- หลักสูตรต้องตกลงกันให้ชัดว่า LLL คืออะไรบ้าง ในบริบทของตน
- LLL ต้องเป็น รูปธรรม วัดได้ ประเมินได้
- กิจกรรมผลักดัน LLL ต้องอยู่ในรายวิชา

อ้างอิง: European Commission — 8 key competences for lifelong learning

R 3.5 — New Ideas, Creative Thought, Innovation, Entrepreneurial Mindset

เนื้อเกณฑ์ที่พึงพิจารณา:

ต้องปลูกฝังให้ นศ. ครบทุกด้าน:
1. มีความคิดใหม่ๆ (New Ideas)
2. มีความคิดสร้างสรรค์ (Creative Thought)
3. มีความคิดเชิงนวัตกรรม (Innovation)
4. มีความคิดของการเป็นผู้ประกอบการ (Entrepreneurial Mindset)

ไม่จำเป็นต้องเป็นวิชาเดียวกัน หรือทุกรายวิชา อาจอยู่ใน GE, วิชาในหลักสูตร, หรือบางคาบ แล้วแต่บริบท/เป้าหมายของหลักสูตร

R 3.6 — ปรับปรุง TLA อย่างต่อเนื่อง

เนื้อเกณฑ์ที่พึงพิจารณา:

มีระบบตรวจสอบวิธีสอนว่าสอดคล้อง CLOs และสอดคล้องกับภาคการทำงาน:

  1. ข้อมูลนำเข้า: ต้องมีข้อมูลนำเข้าสำหรับการทบทวน (เช่น มคอ.5, ข้อเสนอแนะจาก TLA ในเทอมที่ผ่านมา)
    - ระบุได้ว่า: ทบทวนเมื่อไหร่ ใครทำ ใช้ข้อมูลอะไร และปรับปรุงอะไรอย่างไร
  2. สอดคล้อง CLOs และ ตรงตามความต้องการของภาคอุตสาหกรรม
  3. ต้องมี บุคคลที่ 3 เข้ามาร่วมทบทวน (ตรวจโดยคนอื่นด้วย)
  4. ดำเนินการทุกวิชา

Addendum (สำหรับหลักสูตรบัณฑิตศึกษา)

  • Criterion e: "Teaching and Learning Activities" สำหรับ research component อาจรวมถึง discussion/consultation กับ supervisor และ activities/assessment tasks ที่เกี่ยวกับ progress monitoring

ความสัมพันธ์กับ Criterion อื่น

  • R3.3 Active Learning → R4.1 Assessment หลากหลาย (ต้องสอดคล้องกัน)
  • R3.6 ปรับปรุง TLA → ต้องใช้ข้อมูลจาก มคอ.5 (R4.7) และ Feedback ภาคการทำงาน (R2.7)
  • R3.1 Educational Philosophy → ต้องสะท้อนออกมาใน R3.2-R3.5

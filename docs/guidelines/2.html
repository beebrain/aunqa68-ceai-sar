Criterion 2 — Programme Structure and Content

Criterion 2 — Programme Structure and Content

ที่มา: เอกสารอบรม ผศ.ดร.ขวัญดาว (25 มี.ค. 2569)

Requirements Overview

ข้อ ข้อกำหนด (สรุป)
2.1 Programme spec และ Course spec ครอบคลุม ทันสมัย สื่อสารกับ SHs
2.2 หลักสูตรออกแบบแบบ Constructive Alignment กับ PLOs (OBE + BCD)
2.3 การออกแบบหลักสูตรนำ Feedback จาก SHs มาใช้
2.4 แต่ละรายวิชาช่วยผลักดัน PLOs อย่างชัดเจน (Curriculum Mapping คุณภาพ)
2.5 โครงสร้างรายวิชาเป็นระบบ ลำดับจาก Basic → Intermediate → Advanced และมีการ Integrate
2.6 มีทางเลือกให้นักศึกษาเลือกเรียน Major/Minor หรือวิชาเลือก
2.7 หลักสูตรได้รับการทบทวนเป็นระยะตามขั้นตอน ทันสมัย และเกี่ยวข้องกับภาคการทำงาน

R 2.1 — Programme Spec และ Course Spec

เนื้อเกณฑ์ที่พึงพิจารณา:

Programme spec (สื่อโฆษณาหลักสูตร: website, แผ่นพับ, social, โปสเตอร์)
1. Comprehensive — ข้อมูลครบถ้วนตามข้อกำหนด:
- Awarding body, Teaching institution, Accreditation details
- Name of award, Programme title
- Expected learning outcomes
- Admission criteria, Benchmark reports
- Programme structure (levels, credits), Date of writing
2. Up-to-date — ข้อมูลในทุกสื่อตรงกัน ตรงกับ มคอ.2 ล่าสุด
3. Made available and communicated — สื่อสารกับ SHs ทุกกลุ่มในรูปแบบที่เข้าถึงง่าย

Course spec (มคอ.3 / Course Syllabus)
1. Comprehensive — ข้อมูลครบ: Course title, prerequisites, CLOs, Teaching/Learning/Assessment methods, Course outline, Assessment details, Date written/revised
2. Up-to-date — ข้อมูลทุกสื่อตรงกัน
3. Made available and communicated — สื่อสารกับ SHs ที่เกี่ยวข้อง (นศ.ที่ลงเรียน, ผู้สอนทุกคน)

R 2.2 — Constructive Alignment (OBE + BCD)

เนื้อเกณฑ์ที่พึงพิจารณา:

  • วิธีออกแบบหลักสูตรต้องมาจาก OBE และ Backward Curriculum Design (BCD)
  • ต้อง Constructive Aligned: PLO → รายวิชา → ลำดับการเรียน → CLOs → ATs (Assessment Tasks) → LAs (Learning Activities) → CLRs (Course Learning Results) สอดคล้องเชื่อมโยงกัน
  • อ้างอิง: Biggs, J. (2014). Constructive alignment in university teaching.

R 2.3 — Feedback จาก SHs ในการออกแบบหลักสูตร

เนื้อเกณฑ์ที่พึงพิจารณา:

  • กำหนด SHs ที่ต้องเก็บ Feedback (ผู้ให้ข้อมูลต้องเคยสัมผัสกับหลักสูตรของเรา เพื่อบอกได้ว่าดีหรือไม่ดีอย่างไร)
  • กำหนดวิธีเก็บ Feedback จาก SHs แต่ละกลุ่ม
  • นำ Feedback มาใช้ในการออกแบบหลักสูตร (ส่วนใดก็ได้ที่ ไม่ใช่ PLOs): เช่น ทรัพยากร เครื่องมือ อาชีพ

⚠️ สำคัญ: R1.4 = requirement/needs (จาก SHs เพื่อสร้าง PLOs)
R2.3 = feedback (จากผู้ที่เคยสัมผัสหลักสูตร เพื่อปรับปรุงส่วนอื่น)
ต่างกัน!

R 2.4 — แต่ละรายวิชาช่วยผลักดัน PLOs (Curriculum Mapping คุณภาพ)

เนื้อเกณฑ์ที่พึงพิจารณา:

  • ดูคุณภาพของ Mapping (ไม่ใช่แค่มีตาราง แต่ต้องมีคุณภาพ)
    1. ดู รายวิชาบังคับเป็นหลัก
    2. ดูภาระความรับผิดชอบ (จุดดำ = ผลักดันหลัก, จุดขาว = ผลักดันรอง) ทั้งเชิงปริมาณและคุณภาพ
  • แนวนอน: แต่ละรายวิชาช่วยผลักดัน PLOs อะไรบ้าง (ดูรายวิชา)
  • แนวตั้ง: แต่ละ PLO มีรายวิชาใดช่วยผลักดัน (ดู PLO)
  • หลักสูตรต้องอธิบายได้ว่าแต่ละวิชากำลังผลักดัน PLOs อะไร (ดูสาระรายวิชาสอดคล้องกัน)

R 2.5 — โครงสร้างรายวิชาเป็นระบบ ลำดับ และ Integrate

เนื้อเกณฑ์ที่พึงพิจารณา:

  1. โครงสร้างหลักสูตรเหมาะสมตาม มคอ.2 (หมวด GE กี่หมวด วิชาเอกบังคับ/เลือก/เลือกเสรี กี่วิชา กี่หน่วยกิต)
  2. ลำดับรายวิชา: Basic → Intermediate → Specialised (ตาม Learning Level)
  3. Integration: มีรายวิชา/กิจกรรมที่บูรณาการความรู้จากรายวิชา Basic, Intermediate, Advanced เข้าด้วยกัน

R 2.6 — ทางเลือกสำหรับนักศึกษา (Major/Minor/เลือกเสรี)

เนื้อเกณฑ์ที่พึงพิจารณา:

  • มีทางเลือกให้ นศ. เลือกเรียนตามความสนใจ (เลือกโดยเสรี)
  • วิชาเลือกช่วยต่อยอด ขยายผล ส่งเสริมสาขาหลัก

R 2.7 — ทบทวนหลักสูตรเป็นระยะตามขั้นตอน

เนื้อเกณฑ์ที่พึงพิจารณา:

  1. กำหนด ระยะเวลาชัดเจน ที่จะทบทวน
  2. ระบุ ขั้นตอน / กระบวนการ / ผู้รับผิดชอบ
  3. ข้อมูลที่ใช้ทบทวนต้อง เป็นปัจจุบัน (ต้องมีข้อมูลภาคการทำงานหรือความต้องการของสังคม)
  4. ต้องเป็น Formal analysis
  5. ใช้ PDCA: ปรับปรุงย่อย (รายปี) หรือปรับปรุงใหญ่ (รอบ 5 ปี ตาม สป.อว.)

Addendum (สำหรับหลักสูตรบัณฑิตศึกษา)

  • Criterion c: "course" ในบัณฑิตศึกษาหมายถึง programme activities/assessment tasks ที่เกี่ยวกับ progress monitoring และ academic performance review
  • Criterion d: "constructive alignment" สำหรับ research component คือการ mapping ระหว่าง PLOs กับ activities/assessment tasks ที่เกี่ยวกับ progress review ของนักศึกษาปริญญาเอก

ความสัมพันธ์กับ Criterion อื่น

  • R2.2 (Constructive Alignment) → ต้องสอดคล้องกับ R3 (TLA) และ R4 (Assessment)
  • R2.3 Feedback ≠ R1.4 Needs
  • R2.7 ทบทวนหลักสูตร → R3.6 ทบทวน TLA → R4.7 ทบทวน Assessment (ต้องมีระบบทั้งหมด)

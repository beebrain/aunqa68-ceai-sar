ประกาศ มรอ. — ปรัชญาการจัดการศึกษา IEL (25 ส.ค. 2566)

summary

ประกาศมหาวิทยาลัยราชภัฏอุตรดิตถ์

เรื่อง ปรัชญาการจัดการศึกษา มหาวิทยาลัยราชภัฏอุตรดิตถ์

ออกโดย: รองศาสตราจารย์ ดร.สุภาวิณี สัตยาภรณ์ อธิการบดีมหาวิทยาลัยราชภัฏอุตรดิตถ์
วันที่ประกาศ: 25 สิงหาคม พ.ศ. 2566
URL: https://fms.uru.ac.th/fms/filepdf/phylo.pdf
รหัสหลักฐาน AUN-QA: AUNQA-3-1

อำนาจอ้างอิง

  • มาตรา 31(1) แห่ง พ.ร.บ.มหาวิทยาลัยราชภัฏอุตรดิตถ์ พ.ศ. 2547
  • ประกาศกระทรวงการอุดมศึกษา วิทยาศาสตร์ วิจัยและนวัตกรรม เรื่อง กำหนดปรัชญาการอุดมศึกษาไทยและระบบอุดมศึกษาใหม่
  • ประกาศคณะกรรมการมาตรฐานการอุดมศึกษา เรื่อง รายละเอียดผลลัพธ์การเรียนรู้ตามมาตรฐานคุณวุฒิระดับอุดมศึกษา พ.ศ. 2565
  • มติคณะกรรมการบริหารงานวิชาการ ครั้งที่ 7/2566 (10 ก.ค. 2566)
  • มติคณะกรรมการสภาวิชาการ ครั้งที่ 6/2566 (21 ก.ค. 2566)

ปรัชญาการจัดการศึกษา

"สร้างผลลัพธ์การเรียนรู้ด้วยประสบการณ์เชิงบูรณาการ"
Integrated Experiential Learning (IEL)

คำนิยามสำคัญ (4 คำ)

1. ผลลัพธ์การเรียนรู้ (Learning Outcomes)

ผลที่เกิดขึ้นแก่ผู้เรียนผ่านกระบวนการเรียนรู้ที่ได้จากการศึกษา ฝึกอบรม หรือประสบการณ์ที่เกิดขึ้นจากการฝึกปฏิบัติหรือการเรียนรู้จริงในที่ทำงานระหว่างการศึกษา

ประกอบด้วย 4 ด้าน:
| ด้าน | ภาษาอังกฤษ |
|------|-----------|
| ความรู้ | Knowledge |
| ทักษะ | Skills |
| จริยธรรม | Ethics |
| ลักษณะบุคคล | Character |

2. ประสบการณ์เชิงบูรณาการ (Integrated Field Experiences)

กระบวนการเรียนรู้ที่เกิดจากการเรียนรู้และบูรณาการทางวิชาการ วิชาชีพ การฝึกปฏิบัติระหว่างการศึกษาและการฝึกประสบการณ์ เพื่อพัฒนาตนเอง พัฒนาท้องถิ่น สู่การเรียนรู้ตลอดชีวิต

3. ผลลัพธ์การเรียนเฉพาะด้าน (Specific Learning Outcomes — SLOs)

ผลที่เกิดกับผู้เรียนผ่านกระบวนการเรียนรู้รายวิชาเฉพาะด้านของแต่ละหลักสูตร กระบวนการศึกษา การฝึกอบรม การฝึกปฏิบัติหรือการเรียนรู้จริงระหว่างการศึกษา การฝึกเป็นนักคิด นักจัดการ รวมถึงการฝึกประสบการณ์ในสถานประกอบการ

4. ผลลัพธ์การเรียนรู้ทั่วไป (Generic Learning Outcomes — GLOs)

มุ่งพัฒนาผู้เรียนให้เป็นผู้ใฝ่เรียน แสวงหาความรู้เพื่อพัฒนาตนเอง สร้างความรู้คู่คุณธรรม เคารพคุณค่าและศักดิ์ศรีความเป็นมนุษย์ มีความรับผิดชอบต่อสังคมและสิ่งแวดล้อม เป็นพลเมืองดิจิทัลและพลเมืองเข้มแข็ง

การเชื่อมโยงกับ PLO และ CLO ของหลักสูตร CE&AI

IEL Components การสะท้อนใน CE&AI
Learning Outcomes (4 ด้าน) PLO1–PLO3 = Knowledge/Skills; PLO4 = Ethics; PLO5 = Character/LLL
Integrated Field Experiences IEL ระบุในทุก CLO ของมคอ.3 ทุกวิชา; สัปดาห์ 10 ของ 7015101 = IEL community workshop
Specific Learning Outcomes (SLOs) CLO รายวิชา 33 วิชา
Generic Learning Outcomes (GLOs) PLO4 (จริยธรรม), PLO5 (เรียนรู้ตลอดชีวิต/พลเมืองดิจิทัล)

  2. Gap ที่เหลือ (R3.1): ต้องแสดงว่า "สื่อสารปรัชญาไปยัง SHs" อย่างไร — เช่น ปฐมนิเทศ, เว็บไซต์, คู่มือนักศึกษา

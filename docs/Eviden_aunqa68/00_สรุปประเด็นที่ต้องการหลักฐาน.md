## ส่วนที่ 2: เกณฑ์ 8 Criteria

> **[ต้องการหลักฐานเพิ่มเติม: ส่วนที่ 2 ทั้งหมดเป็นฟอร์มเปล่า ยังไม่มีข้อมูลผลดำเนินงานจริง]**

### ข้อมูลหลักฐานที่ต้องเตรียมต่อ Criteria แต่ละข้อ

| Criterion | ประเด็นที่ต้องการ |
|-----------|------------------|
| **Criterion 1** (Expected Learning Outcomes) | หลักฐานว่า PLO ได้สื่อสารให้ผู้มีส่วนได้ส่วนเสียทราบ, หลักฐานการสำรวจความต้องการของผู้ใช้บัณฑิต, หลักฐานว่า PLO สอดคล้องกับ AUN-QA taxonomy |
| **Criterion 2** (Programme Structure) | หลักฐานการออกแบบหลักสูตร (Constructive Alignment), หลักฐานการรับฟังความคิดเห็นจากผู้มีส่วนได้ส่วนเสีย, หลักฐานการทบทวนหลักสูตรเป็นระยะ |
| **Criterion 3** (Teaching and Learning) | หลักฐานปรัชญาการศึกษาถูกสื่อสารและสะท้อนในกิจกรรมการเรียนรู้, หลักฐาน active learning, หลักฐานการส่งเสริมทักษะการเรียนรู้ตลอดชีวิต |
| **Criterion 4** (Student Assessment) | หลักฐานมี rubric, marking scheme, timeline, ระเบียบอุทธรณ์, หลักฐานว่าใช้วิธีการประเมินหลากหลาย, หลักฐานการให้ feedback และการปรับปรุง |
| **Criterion 5** (Academic Staff) | หลักฐานแผนการรับอาจารย์ (succession, promotion), หลักฐานการวัดภาระงาน, หลักฐานการประเมินสมรรถนะ, หลักฐานการพัฒนาวิชาชีพ |
| **Criterion 6** (Student Support) | หลักฐานนโยบายรับนักศึกษา, หลักฐานระบบติดตามความก้าวหน้า, หลักฐานกิจกรรมเสริมหลักสูตร |
| **Criterion 7** (Facilities) | หลักฐานห้องเรียน/ห้องปฏิบัติการเพียงพอ, หลักฐานระบบอินเทอร์เน็ต/IT, หลักฐานห้องสมุดดิจิทัล, หลักฐานความปลอดภัย |
| **Criterion 8** (Output and Outcomes) | **ยังไม่มีผลจริง** — เนื่องจากเป็นหลักสูตรใหม่ ต้องระบุเป็น "แผนงาน" หรือ "คาดการณ์" พร้อมแผนดำเนินงาน |

\psection{ผู้ไม่มาประชุม}
\noindent ไม่มี\par

\startmeeting
\openingchair{ผู้ช่วยศาสตราจารย์ ดร.พิศิษฐ์ นาคใจ}{กล่าวเปิดการประชุมและดำเนินการประชุมตามระเบียบวาระ ดังนี้}

% ============================================================
\agenda{๑}{เรื่องแจ้งเพื่อทราบ}
% ============================================================

ประธานแจ้งให้ที่ประชุมทราบว่า ตามที่หลักสูตรวิศวกรรมศาสตรมหาบัณฑิต สาขาวิชาวิศวกรรมคอมพิวเตอร์และปัญญาประดิษฐ์
ได้เปิดรับนักศึกษารุ่นแรกในภาคเรียนที่ ๑ ปีการศึกษา ๒๕๖๘ เรียบร้อยแล้ว และจากการจัดการเรียนการสอนที่ผ่านมา
พบว่าหลักสูตรจำเป็นต้องเพิ่มอาจารย์ประจำหลักสูตรที่มีความเชี่ยวชาญเฉพาะด้านเพิ่มเติม
เพื่อรองรับการสอนในวิชาที่เกี่ยวข้องกับคณิตศาสตร์ประยุกต์ ฟิสิกส์ วิศวกรรมไฟฟ้า และเทคโนโลยีการผลิต

\noindent\textbf{มติที่ประชุม}\quad ที่ประชุมรับทราบ

% ============================================================
\agenda{๒}{รับรองรายงานการประชุมครั้งที่ ๔/ภาคเรียนที่ ๒/๒๕๖๘}
% ============================================================

ที่ประชุมรับรองรายงานการประชุมครั้งที่ ๔/ภาคเรียนที่ ๒/๒๕๖๘ (วันที่ ๒๗ กุมภาพันธ์ ๒๕๖๙)
โดยไม่มีการแก้ไข

\noindent\textbf{มติที่ประชุม}\quad รับรองรายงานการประชุม

% ============================================================
\agenda{๓}{พิจารณาการปรับปรุงแก้ไขหลักสูตร — ขอเพิ่มอาจารย์ประจำหลักสูตร}
% ============================================================

ประธานนำเสนอร่างแบบฟอร์ม สมอ. ๐๘ การปรับปรุงแก้ไขหลักสูตรวิศวกรรมศาสตรมหาบัณฑิต
สาขาวิชาวิศวกรรมคอมพิวเตอร์และปัญญาประดิษฐ์ ฉบับปี พ.ศ. ๒๕๖๘
เพื่อขอเพิ่มอาจารย์ประจำหลักสูตรจำนวน ๓ คน โดยเหตุผลในการปรับปรุง คือ
หลักสูตรต้องการอาจารย์ที่มีความเชี่ยวชาญหลากหลายมากขึ้น ได้แก่ ด้านการจัดการทรัพยากรและสิ่งแวดล้อม
ด้านวิศวกรรมไฟฟ้าและระบบควบคุม เพื่อเสริมสมรรถนะของนักศึกษาให้ตรงกับความต้องการตลาดแรงงาน

รายชื่ออาจารย์ประจำหลักสูตรที่ขอเพิ่มเติม มีดังนี้

\begin{thailist}
  \item \textbf{ผู้ช่วยศาสตราจารย์ ดร.พิทักษ์ คล้ายชม} (ลำดับที่ ๘)
  ปริญญาเอก: ปร.ด. การจัดการทรัพยากรธรรมชาติและสิ่งแวดล้อม มหาวิทยาลัยราชภัฏอุตรดิตถ์ พ.ศ. ๒๕๖๔
  \item \textbf{ผู้ช่วยศาสตราจารย์ ดร.รัฐพล ดุลยะลา} (ลำดับที่ ๙)
  ปริญญาเอก: ปร.ด. วิศวกรรมไฟฟ้าและคอมพิวเตอร์ มหาวิทยาลัยมหาสารคาม พ.ศ. ๒๕๖๘
  \item \textbf{ผู้ช่วยศาสตราจารย์ ดร.วีระพล คงนุ่น} (ลำดับที่ ๑๐)
  ปริญญาเอก: ปร.ด. วิศวกรรมไฟฟ้าและคอมพิวเตอร์ มหาวิทยาลัยมหาสารคาม พ.ศ. ๒๕๕๗
\end{thailist}

ที่ประชุมร่วมกันพิจารณาคุณสมบัติของอาจารย์ทั้ง ๓ คน ยืนยันว่าตรงตามเกณฑ์มาตรฐานหลักสูตร กมอ. พ.ศ. ๒๕๖๕
ทั้งด้านวุฒิการศึกษา (ปริญญาเอก) และผลงานวิชาการ ไม่มีความซ้ำซ้อนกับหลักสูตรอื่น

\resolution{ที่ประชุมมีมติเห็นชอบการปรับปรุงแก้ไขหลักสูตรตาม สมอ. ๐๘
เพื่อขอเพิ่มอาจารย์ประจำหลักสูตรจำนวน ๓ คน (ลำดับที่ ๘–๑๐)
และให้นำเสนอต่อคณะกรรมการบัณฑิตศึกษาประจำมหาวิทยาลัย และคณะกรรมการบริหารวิชาการประจำคณะ
เพื่อพิจารณาอนุมัติตามลำดับต่อไป}

% ============================================================
\agenda{๔}{เรื่องอื่น ๆ}
% ============================================================

มอบหมายให้รองคณบดีฝ่ายวิชาการและวิเทศสัมพันธ์ (ผศ.ดร.พิทักษ์ คล้ายชม) ดำเนินการจัดส่ง สมอ. ๐๘
พร้อมเอกสารประกอบการตรวจสอบคุณสมบัติอาจารย์ ไปยังกองบริการการศึกษา
เพื่อดำเนินการตรวจสอบและเสนอต่อระดับที่สูงขึ้นต่อไป

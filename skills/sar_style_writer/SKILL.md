# SKILL — SAR Style Writer

## ชื่อ
`sar_style_writer`

## บทบาท
เขียนคำตอบในรูปแบบ SAR (Self-Assessment Report) ที่เป็นทางการ ตรงเกณฑ์ และตรวจสอบหลักฐานได้

## หลักการเขียน SAR

### 1. โครงย่อหน้าแบบมาตรฐาน
เขียนให้ครบ 4 ส่วน โดยรักษาให้เป็น "ความเรียง" ไม่พึ่ง bullet

| ส่วน | เนื้อหา |
|---|---|
| 1. ตอบตรงเกณฑ์ + บริบท | เปิดย่อหน้าด้วยการ "ตอบคำถามเกณฑ์ย่อย" ทันที + บริบทสำคัญ |
| 2. กลไก/กระบวนการ | อธิบาย "ทำอย่างไร" ให้เห็นวงจร + ผู้รับผิดชอบ |
| 3. ข้อมูลเชิงประจักษ์ | ใส่ตัวเลข/ผลลัพธ์/หลักฐาน + รหัสหลักฐาน |
| 4. ช่องว่าง + แผนปรับปรุง | ข้อจำกัด + แผนพัฒนารอบถัดไป |

### 2. สไตล์ภาษาและน้ำเสียง
- **คำกริยาที่ใช้**: กำหนด, จัดทำ, ประกาศ, สื่อสาร, ทบทวน, ติดตาม, รายงาน, นำผลไปปรับปรุง
- **หลีกเลี่ยง**: คำบรรยายลอย ๆ ที่ไม่ระบุหลักฐาน
- **ใส่แหล่งข้อมูล**: ระบุชื่อหน่วยงาน, ระบบ, หรือรหัสหลักฐาน (AUNQA-X-Y)
- **ภาษา**: ไทยทางการ กระชับ ชัดเจน

### 3. การอ้างหลักฐาน
```
รูปแบบ: "(อ้างอิง AUNQA-X-Y)"
ตัวอย่าง: "มีการประเมินผลตามเกณฑ์ที่กำหนด (อ้างอิง AUN-QA-4-1)"
```

### 4. กรณีข้อมูลยังไม่ครบ
เขียนให้ชัดว่า "ยังไม่มีผลลัพธ์เชิงตัวเลขตามนิยามเกณฑ์" แล้วระบุ "กลไกและแผนเก็บข้อมูล" แทน

## Quick Checklist ก่อนจบแต่ละ Criterion

- [ ] เปิดย่อหน้าแรก "ตอบเกณฑ์ย่อย" หรือยัง
- [ ] มีคำอธิบาย "กลไก/ผู้รับผิดชอบ/ความถี่" หรือยัง
- [ ] มี "ข้อมูลเชิงประจักษ์/ตาราง/รหัสหลักฐาน" หรือยัง
- [ ] มี "ข้อจำกัด + แผนปรับปรุง" ชัดเจนหรือยัง
- [ ] ทั้งหมดเป็นความเรียง ไม่ใช้ bullet โดยไม่จำเป็น

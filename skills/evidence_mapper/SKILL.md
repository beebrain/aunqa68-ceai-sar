# Knowledge Module — Evidence Mapping

## แนวคิด
Evidence Mapping คือ การจับคู่ระหว่างเกณฑ์ (criteria) กับหลักฐานที่ต้องการ เพื่อให้สามารถตรวจสอบความครบถ้วนของข้อมูลได้

## รูปแบบหลักฐานที่พบบ่อย

| รหัส | ชื่อ | ใช้กับ |
|---|---|---|
| POL-001 | ประกาศนโยบาย/ระเบียบ | ระบบ/กลไก |
| REG-001 | ระเบียบภายใน | การดำเนินการ |

## ตารางจับคู่ Criteria → Evidence

| Criterion | Evidence ที่ต้องมี |
|---|---|
| 1. Expected Learning Outcomes | มคอ.2, Stakeholder feedback, รายงานการทบทวน |
| 2. Programme Structure | มคอ.2, Curriculum map, ผลการประเมิน |
| 3. Teaching & Learning | มคอ.3, รายงานการสอน, ผลการประเมิน |
| 4. Student Assessment | มคอ.3, Rubrics, ผลสัมฤทธิ์ |
| 5. Academic Staff | HR records, ประวัติอาจารย์, ผลงานวิจัย |
| 6. Student Support | รายงานให้คำปรึกษา, ระบบติดตาม |
| 7. Facilities | รายงานสภาพอาคาร/อุปกรณ์ |
| 8. Programme Outcome | Achievement data, การวิเคราะห์, แผนปรับปรุง |

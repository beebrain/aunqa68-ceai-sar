# Knowledge Module — OBE Concepts

## ความหมาย
**Outcome-Based Education (OBE)** คือ แนวคิดการจัดการศึกษาที่เน้นผลลัพธ์การเรียนรู้เป็นศูนย์กลาง โดยออกแบบหลักสูตร การสอน และการประเมินจากผลลัพธ์ที่ต้องการ

## คำศัพท์สำคัญ

### ELO/PLO (Expected/Programme Learning Outcomes)
ผลลัพธ์การเรียนรู้ของหลักสูตร แสดงสิ่งที่ผู้เรียนควรบรรลุเมื่อสำเร็จการศึกษา

### CLO (Course Learning Outcomes)
ผลลัพธ์การเรียนรู้รายวิชา ย่อยลงมาจาก PLO แต่ละข้อ

### Curriculum Mapping
การจัดตารางแสดงความสัมพันธ์ระหว่าง PLO, CLO, วิธีสอน, และวิธีประเมิน

### Constructive Alignment
หลักการที่ว่า learning outcomes, teaching activities, และ assessment ต้องสอดคล้องกัน

## Action Verbs ตาม Bloom's Taxonomy

| ระดับ | ตัวอย่างคำกริยา |
|---|---|
| Remember | จำ, ระบุ, จัด |
| Understand | อธิบาย, สรุป, แปลความ |
| Apply | ใช้, แก้ปัญหา, ประยุกต์ |
| Analyze | วิเคราะห์, เปรียบเทียบ, แยกแยะ |
| Evaluate | ประเมิน, ตัดสิน, วิจารณ์ |
| Create | ออกแบบ, สร้างสรรค์, พัฒนา |

## Constructive Alignment Framework

```
ELO/PLO → กำหนดผลลัพธ์ที่ต้องการ
    ↓
Teaching & Learning Activities → จัดกิจกรรมให้บรรลุผล
    ↓
Assessment → ออกแบบวิธีวัดให้ตรงกับผลลัพธ์
    ↓
Achievement → วัดผลการบรรลุของผู้เรียน
    ↓
Improvement → นำผลไปปรับปรุง
```

## PDCA Cycle ใน OBE

- **Plan**: กำหนด PLO/CLO วางแผนการสอนและการประเมิน
- **Do**: ดำเนินการสอนและประเมินผล
- **Check**: ตรวจสอบผลลัพธ์การเรียนรู้
- **Act**: ปรับปรุงหลักสูตรและการสอน

# SKILL — AUN-QA SAR response style (benchmark: Modern Business Management)

แนวปฏิบัตินี้สกัดจากรูปแบบการตอบคำถาม AUN-QA ของ “หลักสูตรการจัดการธุรกิจสมัยใหม่” เพื่อใช้เป็น **skill พื้นฐาน** ในการเขียน SAR ของหลักสูตรอื่น (เช่น CE&AI) ให้มีจังหวะการอธิบายที่ชัดเจน อ่านเป็นความเรียง และตรวจสอบหลักฐานได้

## 1) โครงย่อหน้าแบบมาตรฐาน (ใช้กับทุก criterionsub)

เขียนให้ครบ 4 ส่วน โดยรักษาให้เป็น “ความเรียง” ไม่ต้องพึ่ง bullet

1. **ตอบตรงเกณฑ์ + บริบท**: เปิดย่อหน้าด้วยประโยคที่ “ตอบคำถามเกณฑ์ย่อย” ทันที และระบุบริบทสำคัญของหลักสูตร (เช่น ปีหลักสูตร, จำนวนผู้เรียน, ช่วงรายงาน) ที่ทำให้ผู้อ่านตีความข้อมูลได้ถูกต้อง
2. **กลไก/กระบวนการ**: อธิบาย “ทำอย่างไร” ให้เห็นวงจร (กำหนด → สื่อสาร → ดำเนินการ → ติดตาม → ปรับปรุง) พร้อมระบุหน่วยงาน/ผู้รับผิดชอบ
3. **ข้อมูลเชิงประจักษ์**: ใส่ “ตัวเลข/ผลลัพธ์/หลักฐาน” เท่าที่มี และชี้ว่าจะอยู่ในตารางใด (หรือหลักฐานรหัสใด) เพื่อให้ตรวจสอบย้อนกลับได้
4. **ช่องว่าง + แผนปรับปรุง**: ปิดท้ายด้วยข้อจำกัดของข้อมูล/สิ่งที่ยังไม่ครบ และบอกแผนการเก็บข้อมูล/แผนพัฒนาในรอบถัดไปอย่างกระชับ

## 2) สไตล์ภาษาและน้ำเสียง

- ใช้คำกริยาที่ชี้ “การจัดการ” มากกว่าคำบรรยายลอย ๆ เช่น “กำหนด/จัดทำ/ประกาศ/สื่อสาร/ทบทวน/ติดตาม/รายงาน/นำผลไปปรับปรุง”
- ให้ความสำคัญกับ “ความตรวจสอบได้” มากกว่าคำสวย: ระบุแหล่งข้อมูล (REG, HR, ARIT, academic.uru ฯลฯ) และระบุรหัสหลักฐาน AUNQA-X-Y เมื่อกล่าวถึงเอกสาร
- หลีกเลี่ยงการสลับไทย–อังกฤษถี่เกินจำเป็น: หากต้องใช้คำอังกฤษ ให้ใส่วงเล็บครั้งแรก แล้วใช้คำไทยหรือคำทับศัพท์ที่สม่ำเสมอ
- รักษาจังหวะย่อหน้า: 1 ย่อหน้า = 1 แนวคิดหลัก และเชื่อมด้วยคำเชื่อมที่เป็นเหตุเป็นผล (ดังนั้น/ทั้งนี้/โดย/อย่างไรก็ตาม)

## 3) การอ้างหลักฐานแบบ “อ่านแล้วตามได้”

- เมื่อกล่าวถึงหลักฐาน ให้ใช้รูปแบบ **“(อ้างอิง AUNQA-X-Y)”** หรือระบุในประโยคว่า “หลักฐานประกอบคือ AUNQA-X-Y …”
- หากเป็นหลักฐานที่เป็นลิงก์ ให้แนบ `\\url{...}` ในตารางภาคผนวก และให้ข้อความในเนื้อหา “ชี้กลับ” ไปยังรหัสหลักฐานนั้น
- หากข้อมูลยังไม่มี (เช่น ยังไม่มีบัณฑิต) ให้เขียนให้ชัดว่า “ยังไม่มีผลลัพธ์เชิงตัวเลขตามนิยามเกณฑ์” และระบุ “กลไกและแผนเก็บข้อมูล” แทน

## 4) รูปแบบส่วน “รายการหลักฐาน” (แทน bullet)

แทนการลิสต์แบบ itemize ให้ใช้ความเรียง เช่น

> หลักฐานประกอบ Criterion~X ประกอบด้วย AUNQA-X-1 …, AUNQA-X-2 … และ AUNQA-X-3 …

## 5) Quick checklist ก่อนจบแต่ละ Criterion

- เปิดย่อหน้าแรก “ตอบเกณฑ์ย่อย” หรือยัง
- มีคำอธิบาย “กลไก/ผู้รับผิดชอบ/ความถี่” หรือยัง
- มี “ข้อมูลเชิงประจักษ์/ตาราง/รหัสหลักฐาน” ให้ตรวจได้หรือยัง
- มี “ข้อจำกัด + แผนปรับปรุง” ชัดเจนหรือยัง
- ทั้งหมดเป็นความเรียง ไม่ใช้ bullet โดยไม่จำเป็น
